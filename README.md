# kubeplugin

## Installation

```bash
curl -LO https://raw.githubusercontent.com/vhula/kubeplugin/main/scripts/kubeplugin
chmod +x ./kubeplugin
sudo mv ./kubeplugin /usr/local/bin/kubectl-kubeplugin
```

## Usage

```bash
kubectl kubeplugin node <namespace>
kubectl kubeplugin pod <namespace>
```
